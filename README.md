# Fuzzy Logic and Hybrid Intelligent System

## Q1: Washing Machine Fuzzy System

Inputs:
- Dirt Level (Low, Medium, High)
- Load Size (Small, Medium, Large)

Output:
- Wash Time (Low, Medium, High)

Description:
A fuzzy logic system is designed using MATLAB to determine washing machine cycle time based on dirt level and load size.

## Q2: Hybrid System (ANFIS)

Inputs:
- Attendance
- Assignment Marks
- Test Marks

Output:
- Performance Level (Poor, Average, Good)

Description:
ANFIS combines fuzzy logic and neural networks to predict student performance.

## Tools Used
- MATLAB Fuzzy Logic Toolbox
- MATLAB ANFIS
